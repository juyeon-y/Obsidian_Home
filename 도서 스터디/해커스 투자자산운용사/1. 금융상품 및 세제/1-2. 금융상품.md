### 정기성 상품 및 시장실세금리 상품

| 상품                                      | 설명                                                                                                                                                                                                                                                             |
| --------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 정기예금                                    | - 일정 금액을 약정한 기간 동안 예치하고 만기 시 원금과 이자를 받는 상품. 예치기간을 미리 정하여 일정 금액을 예치하고 기간 만료 전에는 원칙적으로 지급청구를 할 수 없도록 되어있는 저축성이 가장 강한 기한부 예금                                                                                                                                      |
| 주가지수연동 정기예금(ELD, Equity-Linked Deposit) | - 주가지수 상승률에 연동하여 사전에 약정한 금리를 지급하는 정기예금의 일종. 정기예금과 파생상품이 결합된 상품으로, 주가지수(KOSPI, S&P500 등)나 특정 주식의 움직임에 따라 수익이 결정<br>- 단, ELD는 `예금자보호에 해당되고, 원금보장형`이라는 특성이 있음 (ELD, ELS, ELF 등은 같은 구조를 갖고 있지만 ELD는 기초자산이 예금)                                                      |
| 양도성예금증서(CD, Certificate of Deposit)     | - 은행이 발행하는 정기예금증서. 정기예금에 양도성을 부여한 것으로 무기명식, 할인식으로 발행<br>- 가입기간 : 30일 이상 제한 없음(최장만기는 제한 없고 최단만기만 제한)<br>- `만기 전 중도해지 불가능(단, 현금화를 위해서 양도가 가능)`                                                                                                                   |
| 환매조건부 채권(RP, Repurchase Agreement)      | - 금융기관이 보유한 채권을 일정 기간 후 되사는 조건으로 고객에게 판매하는 상품. 일정기간 경과후에 사전에 정한 매매가격으로 다시 매수/매도할 것을 조건으로 하는 채권매매 방식<br>- 30일 이내 중도환매 시 당초 약정금리보다 낮은 금리 적용                                                                                                                      |
| 표지어음                                    | - 은행이 할인하여 보유하고 있는 상업어음, 매출채권 등을 분할하거나 통합하여 은행을 지급인으로 하여 새롭게 발행하는 어음. 기존의 어음의 상환 책임을 은행이 맡고, 그에 대한 책임을 대리<br>- 이자지급 : 할인식으로 지급하며 만기 후에는 이자 지급하지 않음<br>- 기명식어음으로 `만기 전 중도해지가 불가능하나 배서에 의한 양도 가능`<br>- 가입기간 : 원어음의 최장만기일 범위 내<br>- 만기 후의 경과기간에 대해서는 별도의 이자지급이 없음 |

### 금융투자상품
#### (1) 자본시장법상 금융투자상품 유형
- 자본시장법에서는 금융투자상품을 증권과 파생상품으로 나눔
- 의의 : 원본 손실 가능성(투자성)을 부담하면서 이익을 얻거나 손실을 회피할 목적으로 거래상대방에게 현재 또는 장래의 특정 시점에 금전 등을 이전하기로 약정하면서 갖게 되는 권리
1. `증권` : 
	- `원본초과손실 가능성이 없는` 금융투자상품. 
	- 다음 6가지로 분류 : 
		1. 채무증권 : 채권, 공사채 등
		2. 지분증권 : 주식, 출자지분 등
		3. 수익증권 : 집합투자기구(펀드)가 발행하는 증권
		4. 투자계약증권 : 공동 사업 손익을 공유하는 증권
		5. `파생결합증권` : 기초자산의 가격이나 지표에 따라 수익이 결정되는 증권(`파생상품이 아닌 증권에 속함!`)
		6. 증권예탁증권 : 외국 발행 증권을 예탁해 발행한 증권
2. `파생상품` : 
	- `원본초과손실 가능성이 있는` 금융투자상품
	- 시장의 형태에 따른 구분 : 장내 / 장외파생상품
	- 계약의 형태(위험배분형태)에 따른 구분 : 선도형 / 옵션형 파생상품

| 상품 예시  | 자본시장법상 분류 | 비고                       |
| ------ | --------- | ------------------------ |
| 주식     | 지분증권      | 발행 회사의 소유 지분             |
| 채권     | 채무증권      | 공사채, 회사채 등               |
| 펀드     | 수익증권      | 집합투자기구(ELF 포함)           |
| ELS    | 파생결합증권    | 주가지수 등 기초자산 변동에 따라 수익 결정 |
| ELB    | 투자계약증권    | 원금 보장이 되는 파생 결합 채권       |
| 선물, 옵션 | 금융 파생상품   | 기초자산의 미래 가격에 따라 수익 결정    |
| 원유 선물  | 상품 파생상품   | 실물 자산(원유)을 기초로 한 파생상품    |

| 항목     | 장내파생상품               | 장외파생상품                             |
| ------ | -------------------- | ---------------------------------- |
| 거래 장소  | 공식적인 거래소(`국내/해외 포함`) | 거래소가 아닌 개별 금융기관들 또는 양 당사자 간에 직접 거래 |
| 표준화 여뷰 | 표준화된 계약              | 맞춤형 계약 (자유 협상)                     |
| 유동성    | 높음 (거래소에서 거래)        | 낮음 (개별 거래에 의존)                     |
| 위험 관리  | 청산소가 존재, 상대방 위험 낮음   | 상대방 위험 존재 (청산소 없음)                 |
| 거래 상대방 | 거래소에 상장된 다양한 시장 참여자들 | 계약 당사자 간의 직접 협상                    |
| 예시     | 선물, 옵션               | 선도계약, 스왑                           |

### 집합투자기구(펀드)의 개념
#### (1) 집합투자의 정의
- `2인 이상의 투자자로부터 모은 금전` 등을 투자자로부터 일상적인 운용지시를 받지 않으면서 재산적인 가치가 있는 투자 대상 자산을 취득/처분/그 밖의 방법으로 운용하고, 그 `결과를 투자자 또는 기금관리주체에게 배분`하여 귀속시키는 것
#### (2) 집합투자기구의 개요
- 집합투자기구는 펀드, 집합투자증권은 수익증권, 투자회사는 뮤추얼펀드로 분류
- 펀드 운용 자산운용사는 집합투자업 인가를 받은 집합투자업자
- 펀드 판매 판매업자는 집합투자증권의 매매중개업 인가를 받은 투자매매/중개업자
#### (3) 집합투자기구의 구성형태
1. 투자신탁 : 집합투자업자인 위탁자가 신탁업자에게 신탁한 재산을 신탁업자로 하여금 집합투자업자의 지시에 따라 투자/운용하게 하는 집합투자기구
2. 투자회사 : 투자회사, 투자유한회사, 투자합자회사
3. 투자조합 : 투자조합, 투자익명조합
#### (4) 기타 용어 정의
- 사모집합투자기구 : 집합투자증권을 사모로만 발행하고 일반투자자의 총수가 49인 이하인 집합투자기구
- 집합투자증권 : 집합투자기구에 대한 출자지분이 표시된 증권
- 집합투자자총회 : 집합투자기구의 투자자 전원으로 구성된 의사결정기관(ex. 수익자총회, 주주총회, 사원총회, 조합원총회, 익명조합원총회)

| 종류           | 설명                                                                |
| ------------ | ----------------------------------------------------------------- |
| 증권집합투자기구     | - 집합투자재산의 50%를 초과하여 증권에 투자하는 집합투자기구                               |
| 부동산집합투자기구    | - 집합투자재산의 50%를 초과하여 부동산에 투자하는 집합투자기구                              |
| 특별자산집합투자기구   | - 집합투자재산의 50%를 초과하여 특별자산(증권, 부동산 제외)에 투자하는 집합투자기구                 |
| 혼합자산집합투자기구   | - 주된 투자대상 및 최저투자한도 등에 제한이 없어 어떠한 자산이든지 투자배율의 제한 없이 투자가 가능한 집합투자기구 |
| `단기금융집합투자기구` | - `집합투자재산의 전부를 단기금융상품에 투자`하는 집합투자기구                               |

### 단기금융집합투자기구(MMF)
- 의의 : 집합투자재산의 전부를 대통령령으로 정하는 단기금융상품에 투자하는 집합투자기구. 대통령령으로 정하는 방법으로 운용.
- 투자가능 단기금융상품 : 원화로 표시된 자산으로서,
	- `남은 만기가 6개월 이내인 양도성예금증서` (증권의 만기가 아니라 남은 만기로 판단함 주의)
	-  `남은 만기가 5년 이내인 국채증권`, `남은 만기가 1년 이내인 지방채증권`, 특수채증권, 사채권 및 기업어음(단, `환매조건부채권(RP) 매수의 대상으로서 채권을 매수할 경우에는 남은 만기 제한을 받지 않음`)
	- 남은 만기가 1년 이내인 금융기관이 발행/할인/중개/매매/인수하는 어음
	- 금융기관에 대한 30일이내의 단기대출
	- 만기가 6개월 이내인 은행/투자매매업자 등 금융기관에의 예치
	- `다른 MMF(다른 주식형펀드 X)`의 집합투자증권
- 운용방법 : 
	- `증권에만 투자 가능`(부동산, 특별자산, 파생상품 X)
	- `증권의 차입과 대여는 모두 금지` (일반 펀드에서는 증권의 차입은 펀드재산총액의 20%까지, 대여는 보유증권총액의 50%까지 가능)
	- 펀드재산의 `전부를 원화로 표시된 단기금융상품에 투자`해야 함(`외화자산은 투자 불가`)
	- `RP(환매조건부 채권) 매도는 펀드보유 증권총액의 5% 이내`에서 운용
	- 펀드재산의 `잔존 가중평균 만기가 75일 이내`일 것(`개별 자산의 만기는 75일을 넘어도 됨`)
	- `남은 만기가 1년 이상인 국채에 대해서는 펀드재산의 5% 이내로 운용`해야 함
	- 개인용 MMF는 3000억원 미만, 법인용은 5000억원 미만인 경우 추가 설정/설립 및 운용업무 수탁 금지
	- MMF 펀드 재산의 `40% 이상을 채무증권(채권)에 투자`해야 함
	- 신용평가 등급이 `최상위등급 또는 최상위등급의 차하위등급(=차상위등급)인 채무증권에만 투자 가능`
	- 채무증권은 최상위등급은 5%, 최상위등급의 차하위등급은 2%를 초과하여 동일인이 발행한 채무증권에 투자 금지(단, 국채, 정부보증채, 지방채, 특수채는 제외)
	- 집합투자업자는 위험관리기준을 정하고 내부통제제도를 갖추어야 함

### 주가지수연계증권(ELS)
#### (1) ELS
- 자본시장법상 `파생결합증권의 한 종류(파생상품이 아님 증권에 포함!)`로, 주가지수 및 특정 주식의 움직임에 연계하여 사전에 정해진 조건에 따라 조기 및 만기상환율이 결정되는 만기가 있는 증권
- 조기상환 및 투자자의 요청에 의한 `중도상환(중도환매)이 가능`(단, 중도상환수수료 부담)
- 발행방식은 `공모발행, 사모발행 모두 가능`. 공모보다 사모의 비중이 높으며, 원금비보장형 상품이 많은 편
- `원금보장형, 원금비보장형 모두 설계 가능`하며, 원금보장형 ELS를 ELB라고 함
#### (2) 수익구조
1. `Knock-out형` : 투자기간 중 사전에 정해둔 주가수준에 도달하면 확정된 수익으로 조기상환되며, 그 외의 경우에는 만기 시 주가에 따라 수익이 정해지는 구조
2. `Bull Spread형` : 만기시점의 주가수준에 비례하여 손익을 얻되 최대 수익 및 손실이 제한되는 구조
3. `Digital형` : 만기 시 주가가 일정 수준을 상회하는지 여부(상승률과는 무관)에 따라 사전에 정한 두 가지 수익 중 한 가지를 지급하는 구조
4. `Reverse Convertible형` : 미리 정한 하락폭 이하로 주가가 하락하지만 않으면 사전에 약정한 수익률을 지급하며 동 수준 이하로 하락하면 원금손실이 발생하는 구조
5. `Step-down형` : 기초자산 가격이 대폭 하락하여 Knock-in이 발생하지 않은 상황에서, 3~6개월마다 주가가 일정 수준 이상인 경우 특정 약정수익률로 자동 조기상환되는 구조
#### (3) 관련 용어
- 낙아웃배리어 : 주로 원금보장형 ELS에서 상승수익률을 지급하는 한계가격 
	- *낙아웃 : 기초자산 가격이 일정 수준에 도달 시 기존의 수익구조가 사라지는 것*
- 낙인배리어 : 주로 원금비보장형 ELS에서 원금손실이 발생할 수 있는 한계가격
	- *낙인 : 기초자산 가격이 일정 수준에 도달 시 새로운 수익구조가 생기는 것*
- 더미수익 : ELS가 조기상환되지 않고 만기까지 보유했을 때 투자기간 중 낙인을 터치한 적이 없으면 만기에 지급되는 보너스 수익
- 리베이트 : 낙아웃 ELS에서 사전에 정한 낙아웃배리어를 초과하여 상승한 경우에 지급되는 고정수익률
- 참여율 : 기초자산의 가격상승률 또한 하락률에 대하여 수익이 어떤 비율로 참여하는가 하는 비율
- 조기상환 또는 자동조기상환 : 조건 달성 시 의무적으로 자동지급되는 자동조기상환 조건
- 중도상환(중도환매) : 투자자의 요청에 의하여 중도에 상환(환매)하는 절차로 중도상환수수료가 공제됨

### 주식워런트증권(ELW, Equity-Linked Warrant)
#### (1) 의의
- 특정 대상물을 사전에 정한 미래의 시기에 미리 정한 가격으로 사거나 팔 수 있는 권리를 갖는 증권
- 기초자산(주식, 주가지수 등)의 가격 변동에 따라 수익을 얻을 수 있는 파생상품으로, 옵션과 유사한 특성을 가짐
- 옵션과 유사한 방식으로, ELW는 기초자산의 가격 상승(콜워런트) 또는 하락(풋워런트)에서 수익을 추구
- ELW는 증권으로 상장되어 거래소에서 매매. 주식처럼 상장된 증권으로 거래되지만, 파생상품으로서 그 가격은 기초자산의 가격 변동에 의존
#### (2) 경제적 기능
- 다양한 투자수단 제공, 보유주식의 활용도 제고, 증권시장의 가격효율성 증대
#### (3) 특징
- 레버리지효과, 한정된 투자위험, 위험헤지 기능, 시장상황과 무관한 새로운 투자수단, 높은 유동성
#### (4) 위험
- 상품의 복잡성, 높은 투자위험, 자본이득 외에 소득이 없음, `주주가 아니며 회사와 직접 관련이 없음(의결권, 배당청구권과 같은 주주로서의 권리 행사 X)`
#### (5) 가격 구조
- `주식워런트증권 가격 = 행사가치(내재가치, 본질가치) + 시간가치`
	- 행사가치 : 
		- `콜옵션의 행사가치 = Max[(기초자산가격 - 행사가격), 0]`
			- 콜 옵션은 기초자산을 특정 가격(행사가격)에 사는 권리를 제공. 기초자산의 가격이 행사가격보다 높을 때만 실제 가치를 가질 수 있음
			- ex1) 기초자산의 가격이 100원이고, 행사가격이 80원일 경우, 행사가치는 100 - 80 = 20원
			- ex2) 기초자산의 가격이 70원이고 행사가격이 80원이라면, 행사가치는 0원
		- `풋옵션의 행사가치 = Max[(행사가격 - 기초자산가격), 0]`
			- 풋 옵션은 기초자산을 특정 가격(행사가격)에 파는 권리를 제공. 기초자산의 가격이 행사가격보다 낮을 때만 실제 가치를 가질 수 있음
			- ex1) 기초자산의 가격이 70원이고, 행사가격이 80원일 경우, 행사가치는 80 - 70 = 10원
			- ex2) 기초자산의 가격이 90원이고 행사가격이 80원이라면, 행사가치는 0원
	- 시간가치 : 미래 가격변동성에 대한 프리미엄

- 가격결정 요인 :

| 구분   | 자산가격 | 행사가격 | 잔존만기 | 가격변동성 | 이자율 | 배당  |
| ---- | ---- | ---- | ---- | ----- | --- | --- |
| 콜 가격 | +    | -    | +    | +     | +   | -   |
| 풋 가격 | -    | +    | +    | +     | -   | +   |
- 요인과 가격이 같은 방향으로 움직이면 +, 다른 방향이면 -
- 기초 자산 가격이 크게 상승하면 콜 옵션의 가치는 증가하고, 크게 하락하면 풋 옵션의 가치는 증가
- 잔존만기(만기 길 수록 시간가치 증가), 가격변동성이 클수록 옵션 가격은 높아짐
- 배당이 높은 기초 자산은 기초 자산 가격을 낮추는 경향이 있음(배당 지급일에 기초 자산의 가격이 배당액만큼 하락). 배당은 콜 옵션의 가치를 하락시키고, 풋 옵션의 가치를 상승
