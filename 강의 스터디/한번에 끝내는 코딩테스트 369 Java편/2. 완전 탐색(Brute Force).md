
### 완전 탐색이란
- 문제 해결 위해 확인해야 하는 `모든 경우를 전부 탐색`하는 방법
- 단순 for문 사용뿐만 아니라, 재귀함수, `백트래킹` 통해 문제 푸는 상황 포함
- 모든 코테 문제에서 `가장 기본적으로 접근해봐야하는 방식`
- 장점 : 부분점수 얻기 좋음
- 단점 : 전부 탐색하기 때문에 시간 복잡도 일반적으로 높음
